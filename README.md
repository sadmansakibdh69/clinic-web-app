# 🏥 Medicare Clinic AI Web App

A fully deployed AI-powered clinic assistant built with Python, Groq API, and Streamlit.

## 🌐 Live Demo
https://clinic-web-app-hh3y4ofaztnxzbmrjnp3zh.streamlit.app/
## What it does
- 💬 Answers patient questions about the clinic
- 📋 Books appointments automatically
- 🧠 AI understands natural language
- 💾 Saves all appointments

## Tech Stack
- Python
- Groq API (LLaMA 3.3 70B)
- Streamlit
- python-dotenv

## How to run locally

1. Clone this repo
2. Install dependencies:
   pip install groq python-dotenv streamlit
3. Create a `.env` file:
   GROQ_API_KEY=your-key-here
4. Run:
   streamlit run app.py
